#### 쿼리문 작성
  ### DB 연결
  sqlplus sys as sysdba
  
    ##사용자 생성
    create user c##scott identified by tiger;
  
    ##사용자 변경
    alter user c##scott default tablespace system temporary tablespace temp;
  
    ## 권한 부여
    grant connect, resource to c##scott;
  
    ## DB연결
    connect c##scott/tiger;
  

#### 테이블 생성 쿼리문
  ### 테이블 생성에 계정 권한 부여
  sqlplus sys as sysdba
  password: test1234
  
  ## 테이블 권한 부여
