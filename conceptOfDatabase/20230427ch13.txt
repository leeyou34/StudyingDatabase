            13-01: 객체지향 데이터베이스
                 
                 1. 데이터 과학의 필요성: 방대한 데이터 속에 숨겨진 규칙과 패턴을 찾아내 문제 해결에 활용하고 앞으로 벌어질 일을 예측하여 미리 준비
                 
                 2. 데이터 과학의 개념: Data              -> Information -> Knowledge -> Wisdom
                                      (상황에 대한 이해)       (의미)        (통찰력)     (지혜)
                                      
                                      Data(데이터): 관찰하거나 측정하여 수집한 사실이나 값을 의미
                                      Information(정보): 상황에 대한 이해를 바탕으로 목적에 맞게 가공한 것이 정보
                                      Knowledge(지식): 규칙과 패턴을 통해 찾아낸 의미 있고 유용한 정보가 지식
                                      Wisdom(지혜): 지식에 통찰력을 더해 새롭고 창의적인 아이디어를 도출한 것이 지혜다.

            13-02: 빅데이터
                1. 빅데이터의 개념
                   빅데이터를 기존의 데이터베이스가 저장하고 관리할 수 있는 범위를 넘어서는 대규모의 다양한 데이터로 좁게 정의하기로 한다.
                   하지만 최근에는 빅데이터를 대규모의 데이터를 저장 및 관리하는 기술과, 가치 있는 저보를 만들기 위해 분석하는 기술까지도 포함하여 정의한다.
                  
                2. 빅데이터의 특징
                   빅데이터의 특징은 데이터양, 속도 다양성을 의미한다.
                   
                   2-1. 데이터양(Volume): 빅데이터는 테라바이트 TB 단위 이상의 대량 데이터다.
                   2-2. 속도(velocity): 빅데이터는  데이터의 수집과 분석을 정해진 시간 내에 처리해야 한다.
                   2-3. 다양성(variety): 형태의 다양성을 띤다. 정형, 반정형, 비정형 같은 다양한 형태의 데이터를 모두 포함한다.
                   
                   2-4. 가치(value):  수집된 빅데이터에 대한 분석으로 얻은 결과는 문제 해결을 위한 의사 결정에 활용될 만한 유용한 가치를 가지고 있다.
                   2-5. 정확성(veracity): 올바른 분석을 방해하는 데이터를 확인해서 제거하는 가공 작업을 통해 되도록 높은 정확성을 유지하는 것이 중요하다.
                   
                   2-6. 시각화(visualization): 빅데이터의 분석결과를 그림이나 도표로 시각화 해서 표현한다.
                   2-7. 가변성(variablity): 생각, 의견, 감정을 소통하기 위해 생성되는 텍스트 형태의 비정형 데이터를  수집하며. 맥락에 따른 가변성을 가지고 있음을 인식하고
                                            수집과 분석 작업에서 데이터의 원래 의미가 그대로 반영된다.
                                            
                                            
               3. 빅데이터의 기술
                  빅데이터에는 저장 기술, 분석 기술, 표현 기술이 존재한다.
                  
                  저장 기술: 하둡과 NoSQL 이 두가지의 기술은 비정형 데이터를 관리하는데 기존 데이터베이스보다 더 뛰어나다.
                             하둡 -> 대용량 데이터를 분산 처리할 수 있는 자바 기반의 오픈 소스 프레임워크다.
                             NoSQL -> 관계 데이터 모델과 SQL을 사용하지 않는 데이터베이스 시스템이다.(가용성과 확장성에 중덤을 둔다)
                             
                  분석 기술: 통계학이나, 컴퓨터 과학 분야에서 데이터 분석을 하는데 사용하는 데이터 마이닝, 기계 학습, 자연어 처리, 패턴 인식 등의 기술을
                             빅데이터에서 데이터 분석을 하는 데 사용한다.
                             
                             1) 텍스트 마이닝
                             2) 오피니언 마이닝
                             3) 소셜 네트워크 분석
                             4) 군집 분석
                             
                  표현 기술: 데이터 분석을 통해 추출한 의미와 가치를 시각적으로 표현하기 위해서 R을 주로 사용한다.
